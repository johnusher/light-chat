Certainly, here's a revised Arduino C code with the FastLED library to create a disco-esque flashing effect at 120 beats per minute:

```cpp
#include <FastLED.h>

#define LED_PIN     6
#define COLOR_ORDER GRB
#define CHIPSET     WS2812B
#define NUM_LEDS    144

#define BRIGHTNESS  200
#define FRAMES_PER_SECOND 120 // Increase frame rate 

CRGB leds[NUM_LEDS];

void setup() {
  delay(3000); // sanity delay
  FastLED.addLeds<CHIPSET, LED_PIN, COLOR_ORDER>(leds, NUM_LEDS).setCorrection(TypicalLEDStrip);
  FastLED.setBrightness(BRIGHTNESS);
  FastLED.show();  // Initialize all leds to black
}

void loop() {
  DiscoLights();

  FastLED.show();
  FastLED.delay(1000 / FRAMES_PER_SECOND);
}

void DiscoLights() {
  
  // Pick a random LED
  int randomLED = random(NUM_LEDS);
  
  // Pick a random color
  CRGB randomColor = CRGB(random8(), random8(), random8());
  
  // Set the LED to the random Color
  leds[randomLED] = randomColor;

  // Randomly 'turn off' half the LEDs each frame to create a flashing effect
  for (int i = 0; i < NUM_LEDS; i++) {
    if (random8() < 128) leds[i] = CRGB::Black;
  }
}
``` 

In the DiscoLights method, we first select a random LED out of the total number (NUM_LEDS). We then generate a random color using the random8() function for each channel (R, G, B). This color is then applied to the previously selected LED.

To achieve the flashing effect, we go through all LEDs and randomly turn about half of them off (that's why random8() < 128) every frame. Since the loop runs at a frame rate of 120 frames per second, this creates the impression of a rapidly blinking, vibrant disco light.
